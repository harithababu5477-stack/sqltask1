College Database Management using SQL

This project demonstrates the creation and management of a College Database using SQL.
It includes designing database schemas, creating tables, defining primary keys, inserting records, and performing basic table operations.

Project Overview

The goal of this project is to practice SQL fundamental by building a simple college database that stores information about students and staff members.

Features Implemented

1.Creating a database schema (`college`)
2.Creating tables for:
    1.Students
    2. Staff
3.Defining columns with appropriate data types
4.Using PRIMARY KEY constraints
5. Inserting records into tables
6.Renaming tables
7.Practicing SQL DDL and DML commands

Tables Structure

Students Table
1. `stu_id` (Primary Key)
2.`stu_name`
3. `gender`
4.`department`
5.`phone_no`

Staff Table
1.`emp_id` (Primary Key)
2. `name`
3. `gender`
4. `degree`
5.`salary`

Technologies Used
1.SQL
2.MySQL (or compatible SQL database)
3. SQL Workbench / MySQL Workbench

Purpose
This project was created as a beginner-level SQL practice project to strengthen understanding of database design, table creation, and data manipulation command.
